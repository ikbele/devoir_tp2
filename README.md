# Atlas Géographique - Flutter

## Description
Application Flutter permettant de découvrir les pays du monde.  
Affiche la liste des pays avec leurs drapeaux et les détails pour chaque pays (capitale, population, superficie, langues).

## Fonctionnalités
- Page d'accueil avec image de bienvenue
- Liste des pays (nom et drapeau)
- Page de détails pour chaque pays
- Navigation fluide entre les pages
